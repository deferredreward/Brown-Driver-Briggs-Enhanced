II. [שָׁוָה] verb

Pi`el set, place (Biblical Aramaic Hithpa. ᵑ7 שְׁוָא Pa`el set, make (often ᵑ7 = שׁום, שׁית); Thes and others below I. ׳ שׁ, originally set, place, then set together, compare, < weakened from original make even, right, compare Vulgar Arabic II. make (Wahrm, compare also BaES 66)); — Perfect 1 singular שִׁוִּיתִי Ps 16:8 I have set ׳ י before me (לְנֶגְּדִּי); עֵזֶר עַל ׳ שׁ Ps 89:20 I have placed strength upon a hero, compare Ps 21:6 Participle מְשַׁוֶּח רַנְלַי כְּ 2Sam 22:34 setting my feet like hinds = Ps 18:34 (compare ᵑ7 שַׁוִּי כְּ Gen 48:20; Jer 29:22, for שׂים כְּ); — Isa 38:13 see I. שׁוה ; — Imperfect = make, produce, 3 masculine singular פְּרִי יְשַׁוֶּה לוֺ Hos 10:1 (Israel as vine), si vera lectio, see We Now (|| עשׂת פְּרִי Hos 9:16, compare Hos 8:7).
