1. ford, only plural Josh 2:7 ( J E ), Judg 3:28 ; Judg 12:5 ; Judg 12:6 ; Isa 16:2 so also 1Sam 13:7 (for ᵑ0 עָֽבְרוּ) We Dr Kit Löhr; + מַעְבֶּרֶת 1Kgs 7:46 = 2Chr 4:17, see סֻכּוֺת
